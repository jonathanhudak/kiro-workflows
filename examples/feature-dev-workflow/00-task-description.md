# Task: Add User Settings Page

## Feature Request

Add a user settings page to the Next.js application where users can update their profile information, notification preferences, and theme settings.

## Requirements

- Users can update display name, email, and avatar
- Users can toggle email notifications (marketing, product updates, security alerts)
- Users can switch between light/dark/system theme
- Changes are saved via API and show success/error feedback
- Page is accessible from the user dropdown menu in the header
- Must work on mobile (responsive)

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Hook Form + Zod validation
- Prisma + PostgreSQL
- NextAuth.js for authentication
